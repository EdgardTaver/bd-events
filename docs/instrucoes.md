# Instruções de uso para interface do CRUD

Fizemos nossa interface por meio de um bot de Telegram.

Nosso CRUD explora a modelagem de músicas, que faz parte da **extensão** do nosso modelo.

Para utilizar o CRUD, siga estas instruções:

1. Acesse o link: [https://t.me/BD2020EACH_Bot](https://t.me/BD2020EACH_Bot).

2. No chat, envie a mensagem `/start`.

3. Digite a senha `melhorbot`.

4. O bot guiará todo o uso do CRUD deste ponto em diante. Ele apresentará 3 opções de tabela: `Música`, `Playlist` e `Grupo Musical`.

5. Para cada uma das tabelas, você terá as opções de `Criar`, `Buscar`, `Atualizar` e `Deletar`.

6. Basta seguir as instruções do bot para cada uma das operações.

7. **Observação**: entre 01:00 am e 07:00 am, o bot fica desligado e portanto, durante este horário, não é possível interagir com o CRUD.
